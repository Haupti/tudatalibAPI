This is a more complete list of all metadata keywords that can be set in TUdatalib.
LARGE DISCLAIMERS:

1) I do not make any claim of completeness!
2) There are NO format for the value keywords given here! I can not make shure that there are not
   any neccessary formats you have to comply for the value keywords for a successfull upload.
3) Not all of the keywords may be in use for the collection your item is in. This may mean that
   you cannot send the keywords for this item and if you do it may lead to errors or the values will simply be 
   ignorend, but i dont know. Make shure that the keywords are in use in the collection you are 
   uploading to!

If you do not know if a desired keyword is activated in the collection you are uploading to you can 
a) ask the collection administrator
b) try if you can set a value for the keyword "manualy" with the graphical user interface of TUdatalib
   (visit the website and try it)
